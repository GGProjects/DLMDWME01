\section{Data description}\label{data-description}

File name: \textbf{sickness\_table.csv}

Saved in: \textbf{../../01\_data/01\_lake/01\_use\_case\_2/}

The dataset has \textbf{1152} observations of \textbf{8} variables and
is of type \textbf{spec\_tbl\_df, tbl\_df, tbl, data.frame}.

The data structure shows following data types:

\begin{verbatim}
cols(
  ...1 = col_double(),
  date = col_date(format = "%Y-%m-%d"),
  n_sick = col_double(),
  calls = col_double(),
  n_duty = col_double(),
  n_sby = col_double(),
  sby_need = col_double(),
  dafted = col_double()
)
\end{verbatim}

The description of the dataset is taken from the projects basic data
document ``Zusatzinformationen\_DLMDWME01 (Pak 2024)

The file provides data about: Anzahl an Krankenständen, Notrufen,
Ersatzfahrern im Bereitschaftsdienst und Zahl der zusätzlichen
Einsatzfahrer, die aktiviert werden müssen (auf Tagesbasis)

Description of variables

\begin{itemize}
\item
  \ldots1: Zähler
\item
  date: Datum
\item
  n\_sick: Anzahl der Einsatzfahrer, die einen Krankenstand angemeldet
  haben
\item
  calls: Anzahl der Notrufe
\item
  n\_duty: Anzahl der Einsatzfahrer im Dienst
\item
  n\_sby: Anzahl der verfügbaren Ersatzfahrer
\item
  sby\_need: Anzahl der Ersatzfahrer, die aktiviert werden
\item
  dafted: Anzahl der zusätzlichen Einsatzfahrer, die aktiviert werden
  müssen, wenn die Anzahl der Einsatzfahrer im Bereitschaftsdienst nicht
  ausreicht
\end{itemize}
